## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- **Node.js** (v16 or higher) — [Download](https://nodejs.org/)
- **npm** or **yarn**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/SauravkuSingh/Krypto.git
   cd Krypto
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**

   Create a `.env` file in the root directory:
   ```env
   VITE_API_KEY=your_api_key_here
   VITE_API_URL=https://api.coingecko.com/api/v3
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

   Open [http://localhost:5173](http://localhost:5173) in your browser 🎉

## 🎨 Design Philosophy

Krypto embraces **brutalist web design** — bold typography, stark contrast, sharp edges, and unapologetic minimalism. Every element serves a purpose. No fluff, just data.

- **Color Palette** — Pure black, white, and accent green/red for market signals
- **Typography** — Heavy, condensed fonts for maximum impact
- **Layout** — Grid-based, information-dense, trader-focused
